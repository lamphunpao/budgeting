# Lamphun Budget Vision

ระบบแสดงข้อมูลงบประมาณขององค์การบริหารส่วนจังหวัดลำพูน

## คุณสมบัติ

### 1. การติดตามการเบิกจ่ายงบประมาณ
- แสดงข้อมูลโครงการสาธารณะปี 2568
- ติดตามความคืบหน้าการเบิกจ่าย
- แสดงสถิติและกราฟการจัดสรรงบประมาณ

### 2. การเปิดเผยรายจ่ายงบประมาณประจำปี
- **หน้าใหม่: `/budgeting/2569`** - แสดงข้อมูลงบประมาณปี 2569
- Dashboard แสดงภาพรวมงบประมาณ
- ตัวกรองข้อมูลตามประเภทงบ, ประเภทค่าใช้จ่าย, และหมวดแผนงาน
- แสดงสัดส่วนงบประมาณในรูปแบบ Progress Bar
- รายละเอียดโครงการทั้งหมดพร้อมการค้นหา

## โครงสร้างข้อมูลงบประมาณปี 2569

ข้อมูลประกอบด้วย:
- **lvl1**: ประเภทงบ (งบบุคลากร, งบดำเนินงาน, งบลงทุน, งบเงินอุดหนุน, งบกลาง)
- **lvl2**: ประเภทค่าใช้จ่าย (เงินเดือน, ค่าตอบแทน, ค่าใช้สอย, ค่าวัสดุ, ค่าครุภัณฑ์, ฯลฯ)
- **project**: ชื่อโครงการ
- **plan**: หมวดแผนงาน (แผนงานบริหารงานทั่วไป, แผนงานการศึกษา, แผนงานสาธารณสุข, ฯลฯ)
- **budget**: งบประมาณที่จัดสรร

## การติดตั้งและรัน

```bash
npm install
npm run dev
```

## การเข้าถึง

- **หน้าแรก**: `/` - ภาพรวมงบประมาณ
- **โครงการสาธารณะ**: `/public-projects` - ติดตามการเบิกจ่าย
- **งบประมาณปี 2569**: `/budgeting/2569` - รายจ่ายงบประมาณประจำปี
- **รายงาน**: `/reports` - รายงานสรุปงบประมาณ
- **แผนที่โครงการ**: `/map` - แสดงโครงการบนแผนที่
